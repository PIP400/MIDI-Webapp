Desktop Overlay Mode

Activating Floating HUD reduces interface elements for maximum borderless transparency overlay.

Embed this configuration block into Electron's main.js to enable standard desktop overlay transparency, click-through click macros, and always-on-top rendering.

const { app, BrowserWindow } = require('electron');

app.whenReady().then(() => {
  const win = new BrowserWindow({
    width: 500,
    height: 500,
    transparent: true,
    frame: false,
    alwaysOnTop: true,
    webPreferences: { nodeIntegration: true }
  });
  
  // Click-Through & Glass Overlay Integration
  win.setIgnoreMouseEvents(true, { forward: true });
  win.loadURL('app-url');
});
